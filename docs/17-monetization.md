# 17 — Monetization

---

## Model

Guild is sold as a self-hosted software license. The developer brings their own Anthropic API key — Guild incurs zero inference costs. Revenue is purely from software value.

---

## Tiers

### Free
- 2 heroes max
- Private memory only (no shared memory)
- CLI only — no dashboard
- No Telegram integration
- No proactive behaviors

### Pro — $20/month or $180/year
- Up to 8 heroes
- Full memory system (shared + private)
- Local dashboard
- Telegram integration (two-way)
- All proactive behaviors
- Priority support
- License key verified locally — works fully offline after activation

---

## Natural Upgrade Trigger

The free tier is limited to 2 heroes. The core quest chain (implement → test → review) requires a **minimum of 3 heroes**. This is architecturally required by the chain rule — not artificial. Free tier can run individual quests but not the full autonomous chain. This is the natural and honest upgrade trigger.

---

## License Distribution

License key is issued at purchase, stored locally, verified via offline algorithm. No license server. No phone-home. Consistent with self-hosted, privacy-first positioning.

---

## Go-to-Market

Target communities: Hacker News (Show HN), r/programming, r/rust, Claude/Anthropic Discord, Twitter/X dev community.

Launch angle: *"I built a self-hosted multi-agent system that remembers your codebase."*
